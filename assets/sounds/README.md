# Audio Files for Smart Fake Call

## Required Files

Place the following MP3 files in this directory:

1. **ringtone.mp3** - Phone ringtone (plays when "call" comes in)
2. **social_1.mp3** - First social conversation audio
3. **social_2.mp3** - Second social conversation audio  
4. **safety_1.mp3** - First safety conversation audio
5. **safety_2.mp3** - Second safety conversation audio

## Quick Reference

- **Social Mode**: Shows "Mama" as caller, plays social_*.mp3
- **Safety Mode**: Shows "Safety" as caller, plays safety_*.mp3

## Need Help?

See `AUDIO_SETUP_GUIDE.md` in the project root for:
- Where to find free audio files
- How to create your own audio

---

**Note**: The app will work without these files, but audio won't play.
Debug console will show "Error playing ringtone/conversation audio" messages.
